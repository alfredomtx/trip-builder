Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

Authenticate requests to this API's endpoints by sending an Authorization header with the value "Bearer {ACCESS_TOKEN}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting the Auth section of this documentation.

Flight

APIs to search for flights.

Search for flights

requires authentication

Search for flights according to search criteria and return paginated result.

Example request:
curl --request GET \
    --get "http://localhost/api/flights/search?departure_airport=YUL.&arrival_airport=YVR.&departure_date=vel&trip_type=quibusdam&return_date=minima&page_size=20&page=1" \
    --header "Authorization: Bearer {ACCESS_TOKEN}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"departure_airport\": \"tcwygiwdwerhmd\",
    \"arrival_airport\": \"jivnaehhidrpbjxmxxzqqntnbrmjkouzorwbuaefxvafwahxdaqoxwxpltldgyhmgppzmrefjgblqfmpptp\",
    \"departure_time\": \"17:24\",
    \"page_size\": 12
}"

const url = new URL(
    "http://localhost/api/flights/search"
);

const params = {
    "departure_airport": "YUL.",
    "arrival_airport": "YVR.",
    "departure_date": "vel",
    "trip_type": "quibusdam",
    "return_date": "minima",
    "page_size": "20",
    "page": "1",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Authorization": "Bearer {ACCESS_TOKEN}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "departure_airport": "tcwygiwdwerhmd",
    "arrival_airport": "jivnaehhidrpbjxmxxzqqntnbrmjkouzorwbuaefxvafwahxdaqoxwxpltldgyhmgppzmrefjgblqfmpptp",
    "departure_time": "17:24",
    "page_size": 12
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Headers

Query Parameters

departure_airport   string   

The departure airport code. Example: YUL.

arrival_airport   string   

The arrival airport code. Example: YVR.

departure_date   string   

date Date of departure. Format YYYY-MM-DD Example: vel

trip_type   string   

Can be a one-way or round-trip. Example: quibusdam

return_date   string  optional  

date Date of return, required if trip-type is round-trip. Format YYYY-MM-DD Example: minima

page_size   integer  optional  

Size per page. Defaults to 10. Example: 20

page   integer  optional  

Page to view. Example: 1

Body Parameters

departure_airport   string   

Must be at least 2 characters. Example: tcwygiwdwerhmd

arrival_airport   string   

Must be at least 2 characters. Example: jivnaehhidrpbjxmxxzqqntnbrmjkouzorwbuaefxvafwahxdaqoxwxpltldgyhmgppzmrefjgblqfmpptp

departure_time   string  optional  

Must be a valid date in the format H:i. Example: 17:24

page_size   integer  optional  

Must not be greater than 100. Example: 12
